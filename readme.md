# MummyIsland  
  
MummyIsland is a 3D game, Written in Python using Pyopengl and Pygame.  
  
  

## How to run
 1. Install python 3  
 2. install Pyopengl and Pygame  
 3. open cmd
 4. `cd <main-directory>  `
 5. `python glut2.py `
 6. Enjoy the game!
 
## How to play
WASD controlls
space for jumping
shift for running
f1 to start/stop recording
f2 to replay a record
esc to exit game

## Cheat Codes
